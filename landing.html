Predicting Future Emissions of Scotland Using Data Science

Tools: Python, Matplotlib, Scikit-Learn, Seaborn

This project involved analysing historical data to predict future trends.

  • Explored data trends and seasonality using Seaborn and Matplotlib visualizations.
